# Jules' Base of Operations

This repository is configured as a high-performance workspace for Jules (AI Agent) and the user.

## Structure

*   `AGENTS.md`: The "brain" of the operation. Contains the rules and guidelines Jules follows to maintain high quality.
*   `src/`: Source code directory.
*   `tests/`: Test directory.

## Environment

This project uses Python.

### Setup

```bash
pip install -r requirements.txt
```

### Running Tests

```bash
pytest
```
