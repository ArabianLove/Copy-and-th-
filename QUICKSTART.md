# Guida Rapida (Quickstart)

Ecco i comandi principali da lanciare nel terminale per usare i nostri strumenti.

## 1. Controllo Sicurezza (Lockdown)
Per vedere se la nostra macchina è sicura e quali porte sono aperte:

```bash
python run_security_check.py
```

## 2. Eseguire i Test
Per assicurarti che tutto l'arsenale funzioni correttamente:

```bash
pytest
```

## 3. Installare/Aggiornare Strumenti
Se aggiungiamo nuove armi, lancia questo per installarle:

```bash
pip install -r requirements.txt
```
